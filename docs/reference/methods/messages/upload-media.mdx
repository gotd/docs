# `messages.uploadMedia`

Upload a file and associate it to a chat (without actually sending it to the chat) May also be used in a business connection, not by wrapping the query in invokeWithBusinessConnection », but rather by specifying the business connection ID in the business_connection_id parameter.

## Parameters

| Name | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `BusinessConnectionID` | `string` | — | Whether the media will be used only in the specified business connection », and not directly by the bot. |
| `Peer` | [`InputPeerClass`](/docs/reference/types/general/input-peer) | yes | The chat, can be inputPeerEmpty for bots and inputPeerSelf for users. |
| `Media` | [`InputMediaClass`](/docs/reference/types/general/input-media) | yes | File uploaded in chunks as described in files » |
